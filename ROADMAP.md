# Roadmap

This document outlines the near-term roadmap for the `astro-bilingual-knowledge-stack` project.

## v0.1.0 - Minimal Public Template (Current)

- [x] Provide neutral bilingual content examples.
- [x] Provide Astro content schemas and locale helpers.
- [x] Provide basic content, link, image, and SEO check commands.
- [x] Provide minimal GitHub Actions for validation, tests, and build.

## v0.2.0 - Tooling & Documentation Enhancements (Near-term)

- [ ] **Language Configuration Examples:** Add comprehensive guides and code samples demonstrating how to extend the template with additional locales beyond English and Chinese.
- [ ] **Documentation Grammar Improvements:** Systematically review and refine documentation across the repository to ensure professional, clear, and grammatically correct English and Chinese texts.
- [ ] **Smoke Test Suite Consolidation:** Enhance the smoke testing tools to run declarative route verification checks against pre-production or production deployments, reducing reliance on manual checks.
- [ ] **Issue and PR Maintenance Workflow:** Establish issue templates, PR review checklists, and automated triaging rules to streamline repository maintenance.

## v0.3.0 - Design, Deployment & Integrations (Near-term)

- [ ] **Dark Mode and Theming Improvements:** Implement a robust CSS-based dark mode toggle and theme customizer, providing clean transitions and premium aesthetics.
- [ ] **Deployment Templates:** Provide ready-to-use infrastructure templates for popular hosting providers (e.g., Cloudflare Pages, Vercel, Netlify, and GitHub Pages).
- [ ] **Verification Tool CLI:** Consolidate check scripts into a single unified maintainer CLI command with clean exit codes and error reports.

## v1.0.0 - Stable Release (Future)

- [ ] Convert the toolkit into an easily installable Astro integration/npm package.
- [ ] Achieve full visual regression testing and 90%+ unit/integration test coverage.
- [ ] Community templates and ecosystem plugins.
