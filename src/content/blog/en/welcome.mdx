Welcome to the Bilingual Knowledge Stack! This post serves as an example of how bilingual content is structured.

## What is this?

This project packages a small workflow for maintaining a bilingual Astro site. It includes:

1. **i18n Routing**: Native support for `/en/` and `/zh/` prefixes.
2. **Quality Gates**: Tools to prevent broken links and missing images.
3. **Automated Logs**: Scripts to automatically generate CHANGELOGs from Git history.

Enjoy building your documentation site!
