# Maintainers

This document lists the maintainers of the `astro-bilingual-knowledge-stack` project and outlines their responsibilities.

## Primary Maintainer

- **Ruoran Guan** / [@ryanguan20001010-star](https://github.com/ryanguan20001010-star)

## Responsibilities

The primary maintainer is responsible for the overall health, direction, and governance of the repository, including:

### 1. Code & Content Review
- Reviewing pull requests for structural and architectural integrity.
- Ensuring code complies with the project's formatting and TypeScript standards.
- Verifying content schemas and i18n configurations align with target locales.
- Guaranteeing all pre-submit quality gates (verify script) pass before merging any contribution.

### 2. Release Management
- Tagging and publishing official releases.
- Maintaining release notes and ensuring deprecation policies are followed.
- Validating build assets and ensuring the public demo is up to date.

### 3. Issue Triage
- Reviewing, tagging, and organizing incoming user issues and bug reports.
- Scoping and labeling issues suitable for external contributors (e.g., `good first issue`).
- Closing stale issues or clarifying requests that do not align with the project's open-source scope.

### 4. Security Coordination
- Reviewing private vulnerability reports.
- Coordinating security fixes and publishing security advisories where necessary.
- Ensuring dependencies are kept up to date and scanned for vulnerabilities.
